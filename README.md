# 🚀 2-DAW: Ejercicios y Tareas

## 📂 Contenido del Repositorio

Actualmente, el repositorio está organizado en base a los siguientes bloques de desarrollo:

*   **🎨 Reto 2: Diseño de Interfaz de Aplicaciones**
    *   *Descripción:* Proyectos enfocados en la maquetación, usabilidad y diseño visual de interfaces web adaptadas al usuario.
*   **💻 Reto 3: Diseño Web Dinámico**
    *   *Descripción:* Desarrollo de sitios web interactivos empleando tecnologías y lógica de programación en el lado del cliente.
*   **⚡ Reto 4: Aplicaciones Web con Bootstrap**
    *   *Descripción:* Creación de aplicaciones web totalmente *responsive* y modernas utilizando el framework Bootstrap para agilizar el diseño.
